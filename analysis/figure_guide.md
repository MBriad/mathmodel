## 图1: 风化关联分析 (fig1a)

**`fig1a_weathering_relations.png`** — 三张并排的堆叠柱状图

**怎么看**：每张图 x 轴是"风化状态"（风化 / 无风化），柱子堆叠的颜色是不同的类别。

**看什么**：
- **左图（风化×类型）**：绿色=高钾，红色=铅钡。风化柱里红色远高于绿色 → 铅钡更容易风化。无风化柱里两色差不多高 → 无风化文物中两类数量接近。
- **中图（风化×纹饰）**：蓝/橙/紫 = 纹饰A/B/C。三根柱子的颜色比例差不多 → 纹饰和风化关系不大。
- **右图（风化×颜色）**：颜色种类多但比例均匀 → 颜色跟风化完全无关。

**结论信号**：左图两根柱子颜色比例明显不同 = 有显著关联。中图和右图比例差不多 = 没显著关联。

## 图2-4: 风化前后化学对比 (fig1b)

### **`fig1b_boxplots.png`** — 10种关键氧化物的箱线图

**怎么看**：每个小图是一种氧化物（如 SiO2）。里面有4个箱子，对应4组数据（高钾/铅钡 × 风化/未风化）。箱子中间的横线是中位数，箱子上下沿是 25% 和 75% 分位数，须线延伸到正常范围，须线外的点是异常值。

**看什么**：
- **SiO2**：高钾-风化（第二个箱子）的箱子在 ~93%，远高于其他三组 → 高钾风化后几乎只剩 SiO2
- **K2O**：高钾-无风化（第一个箱子）在 ~8%，高钾-风化下降到 ~0.5% → K2O 几乎全流失
- **PbO**：铅钡-风化（第四个箱子）在 ~37%，铅钡-无风化（第三个）在 ~24% → PbO 风化后相对富集
- **BaO**：两组铅钡的箱子都在 ~6-10%，位置相似 → BaO 在风化中相对稳定

**结论信号**：同一氧化物的箱子在不同组间位置明显不同 = 风化对该氧化物有显著影响。箱子高度差不多 = 风化不影响该成分。

### **`fig1b_radar.png`** — 四组化学成分雷达图

**怎么看**：14个轴 = 14种氧化物。每条彩色线代表一组数据在各氧化物上的均值。离中心越远 = 含量越高。

**看什么**（所有线条均为实线，半透明填充区分各组）：
- 绿色线（高钾-无风化）：在 K2O 方向有突起，在 SiO2 方向也较高
- 红色线（高钾-风化）：**几乎只在 SiO2 方向突出**，其他方向全部贴近中心 → 高钾风化后成分极度单一
- 蓝色线（铅钡-无风化）：在 PbO、BaO、SiO2 三个方向有突起 → 铅钡的三角特征
- 橙色线（铅钡-风化）：PbO 方向突起更大，SiO2 方向收缩 → 风化让铅更突出、硅更少

**结论信号**：线条形状越不同 = 化学成分差异越大。高钾风化前后的线形差异巨大（风化重塑了全部成分）。

### **`fig1b_diff_bar.png`** — 风化效应差值棒图

**怎么看**：每根棒 = 一种氧化物风化后的含量减去风化前的含量。红色棒 = 风化后增加（向上），绿色棒 = 风化后减少（向下）。棒越长 = 变化越大。

**看什么**：
- **左图（高钾）**：只有一根大红棒（SiO2, +26%），其余全是绿棒 → 高钾风化就是 SiO2 富集 + 其他全流失
- **右图（铅钡）**：大红棒是 PbO (+13%) 和 P2O5 (+3%)，大绿棒是 SiO2 (-20%) → 铅钡风化是 SiO2 流失 + PbO/P2O5 进入

**结论信号**：红绿分布模式完全不同 → 两种玻璃的风化机制本质不同。

## 图5: 风化前后预测对比 (fig1c)

**`fig1c_predictions.png`** — 四张子图

**怎么看**：
- **左上（铅钡-49）**：红色柱（风化后实测）vs 绿色柱（预测风化前）。重点看 PbO 和 SiO2 的差异。
- **左下（高钾-07）**：同样是实测 vs 预测对比。高钾风化后 SiO2 极高 (~93%)，预测回 ~68%。
- **右上（铅钡校正向量）**：横棒图，展示用来校正铅钡风化效应的 Δ 值
- **右下（高钾校正向量）**：同上，高钾的校正值

**看什么**：
- 49号文物：实测 PbO=34%，预测回 23%——实际未风化点是 23% → 预测准确
- 07号文物：实测 SiO2=93%，预测回 68%——风化高钾几乎纯硅

**结论信号**：红绿柱差异大的氧化物 = 风化敏感成分。校正向量图告诉你每种氧化物风化时是"增加"还是"流失"。

## 图6: 分类判别力 (fig2a)

### **`fig2a_discrimination.png`** — 三张子图

**怎么看**：
- **左图（AUC横棒）**：每根棒 = 一种氧化物单独区分高钾/铅钡的能力。越接近1.0越好。蓝色棒 >0.9 = 优秀判别器。
- **中图（PbO-K2O散点）**：每个点 = 一个未风化样品。x=PbO含量，y=K2O含量。绿色=高钾，红色=铅钡。
- **右图（PbO-BaO散点）**：同上，y轴换成 BaO。

**看什么**：
- 左图：只有 PbO 和 BaO 是蓝棒（AUC=1.0）→ 完美判别器。其他氧化物 AUC<0.7 → 单独用不可靠。
- 中图：红点集中在右下（高PbO低K2O），绿点集中在左上（低PbO高K2O）→ 两类完全分离
- 右图：红色全在上方（BaO>3.4%），绿色全在下方（BaO<2.9%）→ **BaO 一刀就能切干净**

**结论信号**：散点图中红绿点没有任何重叠 = 该氧化物组合可以完美分类。AUC=1.0 = 100% 准确率。

### **`fig2a_roc.png`** — ROC曲线

**怎么看**：x轴=假阳性率（误判高钾为铅钡的比例），y轴=真阳性率（正确抓到铅钡的比例）。每条线代表一个分类器。

**看什么**：
- PbO（蓝线）和 BaO（绿线）紧贴左上角 → AUC≈1.0，完美
- LDA（黑色虚线）略低 → 多变量反而不如单变量
- 灰色对角线 = 随机猜测基线

**结论信号**：线越靠近左上角越好。贴着左上角 = 完美分类。

## 图7-8: 亚类聚类 (fig2b)

### **`fig2b_高钾_subclass.png`** 和 **`fig2b_铅钡_subclass.png`**

每张有三幅子图：

**左图（PCA散点）**：每个点 = 一个样品，颜色 = 聚类分配的亚类。x=第一主成分，y=第二主成分。点越近 = 成分越相似。

**看什么**：同色点聚在一起 = 亚类内部成分一致。不同色混在一起 = 亚类分得不好。

**中图（肘部+轮廓）**：蓝线 = 簇内距离（越小越好，但递减），红线 = 轮廓系数（越大越好）。最优 k 是红线最高点。

**看什么**：红线在哪个 k 最高，就选哪个 k。

**右图（雷达图）**：每种颜色 = 一个亚类的平均成分轮廓。看不同亚类在哪个氧化物方向上分开。

**看什么**：
- 高钾：亚类 B（橙色，n=8）几乎只在 SiO2 方向突出 → 这是风化高钾组。其他亚类在 K2O/CaO/Al2O3 方向各有不同。
- 铅钡：亚类 A（蓝色，n=18）PbO 最高、SiO2 最低，亚类 B（橙色，n=5）BaO 极高、CuO 突出，亚类 C（绿色，n=26）SiO2 最高。

**结论信号**：雷达图线条形状明显不同 = 亚类有化学意义。形状相似只是高度不同 = 本质是同一类只是风化程度不同。

## 图9: 未知样品分类 (fig3)

**`fig3_classification.png`** — 两张子图

**左图（PbO-BaO 空间）**：
- 淡色圆点 = 已知训练数据（绿=高钾，红=铅钡）
- 大星号 = 8个未知样品，颜色标注共识分类结果
- 灰色虚线 = BaO=3.14% 决策边界

**看什么**：星号落在虚线以上 = 分类为铅钡，虚线以下 = 高钾。如果星号靠近虚线 = 分类不确定。

**关键看点**：A2（PbO=34.3%, BaO=0）落在右下角——PbO 极高但 BaO=0，跟所有已知样品都不一样。这是异常信号。

**右图（阈值敏感性）**：x轴 = BaO 阈值变化，y轴 = 被分类为铅钡的样品数。红色虚线 = 最优阈值 3.14%。灰色带 = ±20% 扰动范围。

**看什么**：曲线在灰色带内如果平坦（不变）= 分类结果对阈值不敏感 = 稳健。如果在某个值突变 = 有样品正好在阈值附近。

## 图10-11: 关联分析 (fig4)

### **`fig4_correlation_heatmaps.png`** — 四张热力图

**怎么看**：每张是 14×14 的格子矩阵。行和列都是氧化物。格子颜色 = 两个氧化物的相关系数。红色 = 正相关（一个高了另一个也高），蓝色 = 负相关（一个高了另一个低）。颜色越深 = 关系越强。对角线永远是深红色（自己跟自己完全正相关）。

**看什么**：
- **高钾-Pearson（左上）**：SiO2 那一行很多红色 → 在 CLR 空间中，SiO2 与多种微量元素呈正相关。注意 CLR 反映的是相对丰度关系而非绝对含量：风化导致高钾玻璃中其他成分流失、SiO2 相对富集，因此 SiO2 的相对丰度与残留微量元素同步变化——这不是绝对含量的正相关，而是风化残留效应的数学表现。
- **铅钡-Pearson（左下）**：MgO-Al2O3 格子深红，SiO2-Na2O 深红 → 矿物组合共现
- 高钾和铅钡的热力图**整体模式不同** → 两种玻璃的化学关联结构本质不同

**结论信号**：看偏离对角线的深色格子 = 重要的化学关联。比较上下两行（高钾 vs 铅钡）= 看关联模式差异。

### **`fig4_difference_network.png`** — 两张子图

**左图（差异热力图）**：每个格子 = 铅钡的相关系数减去高钾的相关系数。红色 = 铅钡中关联更强，蓝色 = 高钾中关联更强。标注的数字是具体差值，只标注了统计显著的。

**看什么**：
- SiO2-SO2 格子是深蓝色（-1.21）→ 高钾中 SiO2-SO2 强正相关，铅钡中是负相关 = 完全反转
- SiO2-Al2O3 格子是深红色（+0.95）→ 铅钡中正相关，高钾中弱负相关

**右图（关联网络）**：每个圆 = 一种氧化物。连线 = 两者存在强关联（|r|>0.5）。绿线 = 两类都有，红线 = 两类共有且方向一致（负相关），浅绿虚线 = 仅高钾有，浅红虚线 = 仅铅钡有。

**看什么**：
- 绿线 = 两类玻璃共享的化学规律（如 MgO-Al2O3 总是伴生）
- 浅绿线 = 高钾特有的关联（风化驱动）
- 浅红线 = 铅钡特有的关联（原始配方驱动）
- 网络整体结构差异大 → 两种玻璃的化学体系不同

**结论信号**：左右两图的色块模式完全不对称 = 两类玻璃的化学关联网络在统计上显著不同。
