### 采样点命名规则

| 后缀 | 含义 |
|------|------|
| 无后缀 | 文物表面随机采样 |
| 部位1 / 部位2 | 造型不同部位分别采样 |
| 未风化点 | 风化文物表面的未风化区域 |
| 未风化点1 / 未风化点2 | 多个未风化点 |
| 严重风化点 | 取自风化层 |
| 未风化点 + 严重风化点 | 同一风化文物的对比采样 |

### 数据有效性

- 各成分比例累加和应接近100%
- 有效数据范围：85%~105%
- 部分样品总和不在此范围，可能由检测手段限制导致

## 关键分析要点

1. **高钾玻璃 vs 铅钡玻璃**：
   - 高钾玻璃：K2O含量高，PbO和BaO含量低
   - 铅钡玻璃：PbO和BaO含量显著高

2. **风化影响**：
   - 风化会导致易溶成分（Na2O, K2O）流失
   - PbO, BaO等重元素相对富集
   - SiO2含量可能因其他成分流失而相对升高

3. **亚类划分**：
   - 可依据特征氧化物含量进行聚类分析
   - PbO/BaO比值、K2O/SiO2比值等可作为分类指标
