文件 `mypaper.pdf` 是针对 `C题.pdf` 提出的数学建模问题所撰写的参赛论文 ，而其论文格式严格遵循了 `format2022.doc` 中的规范 。  



具体对应关系如下：

### 1. 内容对应：mypaper 是对 C 题的解答



`mypaper.pdf` 完整解决了 `C题.pdf` 中提出的四个核心问题 ：  



- 

  **问题 1：风化规律与预测**   

  

  

  - 

    **对应关系**：`mypaper` 采用了卡方检验、斯皮尔曼相关系数分析风化与纹饰、类型的关系，并建立岭回归分析模型预测风化前的化学成分 。  

    

    

- 

  **问题 2：分类规律与亚类划分**   

  

  

  - 

    **对应关系**：`mypaper` 采用熵值法得出分类规律，并利用 $K\text{-}means++$ 与 BP 神经网络的混合模型对高钾和铅钡玻璃进行亚类划分 。  

    

    

- 

  **问题 3：未知类别鉴别**   

  

  

  - 

    **对应关系**：`mypaper` 组合了 LGBM、逻辑回归、SVC 等七个模型建立集成学习模型（StackingCVClassifier），对附件表单三中的未知样品进行了类型预测 。  

    

    

- 

  **问题 4：关联关系分析**   

  

  

  - 

    **对应关系**：`mypaper` 采用灰色关联分析模型探究不同类别样品群内部化学成分的关联性 。  

    

    

### 2. 格式对应：mypaper 遵循 format2022 规范



`mypaper.pdf` 的排版完全符合 `format2022.doc` 要求的“全国大学生数学建模竞赛论文格式规范” ：  



- 

  **首页结构**：`format2022` 要求首页（电子版第一页）必须为摘要专用页，包含标题、摘要和关键词 。`mypaper` 第一页即为标题“基于多模型集成的古代玻璃制品成分分析与鉴别”、摘要及关键字 。  

  

  

- 

  **正文要求**：`format2022` 规定论文从第四页（纸质版）或第二页（电子版）开始为正文，且不要目录 。`mypaper` 从第二页开始直接进入“1 问题重述”等正文内容，未设目录 。  

  

  

- 

  **附录与代码**：`format2022` 要求在附录中提供支撑材料列表及完整源程序 。`mypaper` 在文末附录中详细展示了共线性诊断图表、亚分类结果、灰色关联结果及完整的 Python 源代码 。
